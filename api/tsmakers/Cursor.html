class tsmakers.Cursor.Cursor(sequence=1, 2, 3, index=None)¶

A cursor.

>>> cursor = tsmakers.Cursor([1, 2, 3])
>>> next(cursor)
1

>>> next(cursor)
2

>>> next(cursor)
3

>>> next(cursor)

>>> next(cursor)
2

>>> cursor.backtrack()

>>> cursor.backtrack()

>>> cursor.backtrack()

>>> next(cursor)
